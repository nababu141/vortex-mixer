💎 What is Diamond Vortex?
Diamond Vortex is a digital asset mixer purpose-built for the **BASE Network**. The protocol ensures transaction privacy by breaking the on-chain link between the deposit address and the destination address.

### 🛡️ Security Pillars:
* **Non-Custodial:** The user remains in control. Funds are processed through ephemeral, secure addresses.
* **Zero-Log Policy:** Our node is configured to automatically wipe all transaction data once the mixing process is confirmed on the blockchain.
* **Secure Tunneling:** The interface communicates with our private node via encrypted tunnels, shielding the user's metadata and origin.

### 🚀 How it Works:
1. **Generation:** Obtain a unique deposit ticket and a secure secret code.
2. **Deposit:** Send your assets to the generated secure address.
3. **Mixing:** The system monitors the blockchain for the required confirmations.
4. **Withdrawal:** Enter your secret code and destination address. Our protocol distributes the funds anonymously.
